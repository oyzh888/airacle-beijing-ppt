## ✅ 如何认领任务

1. 在此文件中找到你要做的任务
2. 把 `_待分配_` 改成你的名字
3. 把 `[ ]` 改成 `[x]` 标记完成
4. Commit message: `chore(todo): assign task XX to YY` 或 `chore(todo): complete task XX`
5. 完成后 push 到 main 分支
